# Python Concepts

This repository contains Jupyter notebooks covering important Python programming concepts.

## 1. Conditions (`Conditions.ipynb`)

The Conditions notebook explains how decision-making works in Python. Conditions allow a program to execute different blocks of code based on whether a statement is true or false.

### Topics Covered
- if statement
- if-else statement
- if-elif-else statement
- Nested conditions
- Logical operators
- Comparison operators

### Purpose
Conditions help control the flow of a program and make it capable of handling different situations and user inputs.

---

## 2. Object-Oriented Programming (`oopss.ipynb`)

The OOPS notebook introduces the Object-Oriented Programming approach in Python. OOP helps organize code into reusable and manageable structures using classes and objects.

### Topics Covered
- Classes and Objects
- Constructors
- Instance Variables
- Methods
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

### Purpose
Object-Oriented Programming improves code reusability, scalability, and maintainability, making it easier to develop large applications.

---

## Learning Outcome

After completing these notebooks, learners will be able to:
- Apply conditional logic in Python programs.
- Create and use classes and objects.
- Understand the core principles of Object-Oriented Programming.
- Write more structured, reusable, and efficient Python code.
